# About the Project

> A consolidated group of github actions for Mystic Framework.

## Introduction

`thedevmystic/github-actions` is a collection of actions designed to make workflows maintainable.
It provides reusable, customizable, and readable workflow actions. These actions include compatibility
checks for languages like `C++` and `Python`, linting, `pre-commit` checks and other useful actions.

## Features

- **Support:** Cross-platform support for Windows, macOS, and Linux.

- **Customizable:** Each action can be customized to suit any need.

- **Readable:** Clear and concise messages and errors. No need to decipher anything!
